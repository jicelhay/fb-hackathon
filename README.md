# fb-hackathon
Repositorio con estructura si se usa Ionic

#Versiones
Cordova 5.4.1
Ionic 1.7.12
node 0.12.2

# Compilación
En el folder donde está config.xml
```sh
$ npm install
$ bower install
```

Para desarrollo en navegador
```sh
$ ionic serve
```

Para compilar y ejecutar aplicación en dispositivo Android
```sh
$ ionic platform add android
$ ionic run android
```
